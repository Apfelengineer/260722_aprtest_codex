# なんでも質問保存箱

質問と回答を、ユーザーが選んだフォルダの `data.csv` に保存するローカル専用アプリです。入力内容を外部へ送信する機能はありません。

## 起動方法

Python 3 が入った macOS / Windows / Linux で動作します。追加ライブラリは不要です。

### macOS

`run.command` をダブルクリックします。初回に確認画面が出た場合は、ファイルを右クリックして「開く」を選んでください。

またはターミナルで次を実行します。

```sh
python3 app.py
```

### Windows

このフォルダでコマンドプロンプトを開き、次を実行します。

```bat
python app.py
```

## 使い方

1. 「フォルダを選択…」で保存先を選びます。
2. 「質問を追加」で質問と回答を入力し、「CSVに保存」を押します。
3. 「質問一覧」で保存済みデータを確認します。

選択したフォルダに `data.csv` が自動作成されます。見出しは「質問」「回答」、文字コードは UTF-8（BOM付き）です。
